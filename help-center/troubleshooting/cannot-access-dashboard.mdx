---
title: Cannot Access Aly Sidi Dashboard
description: Fix dashboard login issues.
---

## Symptoms

- Login page not loading
- Access denied error

## Common Causes

- Incorrect credentials
- Browser cache issue

## Quick Fix

<Steps>
  <Step title="Check your username/password">
    Try resetting your password if unsure.
  </Step>
  <Step title="Clear browser cache">
    Log out, clear cache, then try again.
  </Step>
  <Step title="Still not working?">
    Contact <a href="mailto:support@alysidi.com">support@alysidi.com</a>.
  </Step>
</Steps>
