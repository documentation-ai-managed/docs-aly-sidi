## Overview

TinyEmail supports seamless integrations with popular e-commerce platforms, automation tools, and custom services. You can sync subscriber data, trigger campaigns based on external events, and automate workflows without leaving your dashboard.

Use these integrations to:
- Import customers from Shopify
- Set up webhooks for real-time notifications
- Connect via API to CRMs like HubSpot
- Sync analytics with Google Analytics

## Webhook Setup for Custom Automations

Webhooks let external services push data to TinyEmail in real-time, such as new form submissions or payment confirmations.

### Create a Webhook Endpoint

Follow these steps to set up a webhook receiver.

<Steps>
  <Step title="Generate URL" icon="link">
    In your TinyEmail dashboard, navigate to Integrations > Webhooks and create a new endpoint.
  </Step>
  <Step title="Configure Payload">
    Select events like `subscriber.created` or `campaign.sent`.
  </Step>
  <Step title="Test Delivery">
    Use the test button to verify the endpoint receives data.
  </Step>
</Steps>

<Request tabs="cURL,JavaScript" show-lines="true">
  ```bash
  curl -X POST https://api.example.com/v1/webhooks/tinyemail \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "event": "subscriber.created",
      "data": {
        "email": "new@example.com",
        "name": "New User"
      }
    }'
  ```

  ````javascript
  const response = await fetch('https://api.tinyemail.com/v1/webhooks/tinyemail', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      event: 'subscriber.created',
      data: {
        email: 'new@example.com',
        name: 'New User'
      }
    })
  });
  ````
</Request>

<Response tabs="200,400">
  ```json
  {
    "success": true,
    "subscriber_id": "sub_123abc"
  }
  ```

  ```json
  {
    "error": "Invalid signature",
    "message": "Verify HMAC signature"
  }
  ```
</Response>

## Third-Party App Connections via API

Build custom integrations using TinyEmail's REST API. Authenticate with an API key from your account settings.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField query="list_id" param-type="string" required="false">
  Target subscriber list ID.
</ParamField>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const addSubscriber = async (email, listId) => {
    const response = await fetch(`https://api.example.com/v1/lists/${listId}/subscribers`, {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ email })
    });
    return response.json();
  };
  ```

  ```python
  import requests

  response = requests.post(
      'https://api.example.com/v1/lists/{list_id}/subscribers',
      headers={
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json',
      },
      json={'email': 'user@example.com'}
  )
  ```
</CodeGroup>
